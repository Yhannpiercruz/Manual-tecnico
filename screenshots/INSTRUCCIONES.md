# Instrucciones para Capturas de Pantalla

## Capturas Necesarias

Para completar el manual técnico, toma las siguientes capturas de pantalla del proyecto Renacer:

### 1. accesibilidad.png
- **Ubicación**: Cualquier página del sistema
- **Elemento**: Botón de modo daltónico (ícono de ojo)
- **Tamaño recomendado**: 400x200px
- **Descripción**: Captura del botón de accesibilidad en el header

### 2. header.png
- **Ubicación**: Cualquier página del sistema
- **Elemento**: Barra superior completa
- **Tamaño recomendado**: 1200x100px
- **Descripción**: Header con título del módulo e información del usuario

### 3. menu.png
- **Ubicación**: Cualquier página del sistema
- **Elemento**: Menú lateral (sidebar)
- **Tamaño recomendado**: 300x600px
- **Descripción**: Sidebar con todos los ítems del menú visible

### 4. login.png
- **Ubicación**: Página Index.html
- **Elemento**: Modal de inicio de sesión
- **Tamaño recomendado**: 500x600px
- **Descripción**: Modal de login con campos de usuario y contraseña

## Cómo Tomar las Capturas

### Opción 1: Herramienta de Recorte de Windows
1. Presiona `Windows + Shift + S`
2. Selecciona el área deseada
3. Guarda la imagen en la carpeta `screenshots/`

### Opción 2: Navegador (Chrome/Edge)
1. Abre las DevTools (F12)
2. Presiona `Ctrl + Shift + P`
3. Escribe "screenshot"
4. Selecciona "Capture node screenshot" o "Capture area screenshot"

### Opción 3: Extensión de Navegador
- Usa extensiones como "Awesome Screenshot" o "Nimbus Screenshot"

## Formato de Imágenes

- **Formato**: PNG (preferido) o JPG
- **Calidad**: Alta resolución
- **Fondo**: Incluir el contexto necesario
- **Nombres**: Usar exactamente los nombres indicados arriba

## Ubicación Final

Todas las capturas deben guardarse en:
```
manual-tecnico/screenshots/
```

## Capturas Opcionales (Recomendadas)

Si deseas ampliar el manual, puedes agregar:

- `tabla.png` - Ejemplo de tabla con datos
- `modal.png` - Modal de edición/creación
- `formulario.png` - Formulario completo
- `botones.png` - Conjunto de botones del sistema
- `buscador.png` - Campo de búsqueda en uso
- `dashboard.png` - Vista del dashboard/inicio
- `inventario.png` - Módulo de inventario
- `ventas.png` - Módulo de ventas
- `compras.png` - Módulo de compras

## Notas Importantes

- Las imágenes deben ser claras y legibles
- Evita incluir datos sensibles o reales
- Usa datos de prueba si es necesario
- Mantén un estilo consistente en todas las capturas
- Considera usar el modo claro (no daltónico) para las capturas principales

## Alternativa: Imágenes Placeholder

Si no puedes tomar capturas ahora, el manual funcionará igualmente. Las imágenes aparecerán como placeholders hasta que las agregues.
